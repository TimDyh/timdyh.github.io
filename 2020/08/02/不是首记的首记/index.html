不是首记的首记

发表于 2020-08-02 分类于 随笔 阅读次数：
本文字数： 828 阅读时长 ≈ 1 分钟

趁实习的闲暇时间，断断续续地花了一个多星期，终于把建个人博客这件很早就想做的事情做了。

实际上这并不是笔者的第一个博客。熟悉我的人应该知道，我在大二的暑假就用 Django 手写过一个博客，自己买了服务器和域名，做了备案，从头到尾 DIY，当然除了主题还是直接用的 NexT。虽然在这个过程中学到了不少东西，但有一说一，完全靠自己维护一个博客网站实在太麻烦。隔三差五就 500 错误，当时的我技术还不行，遇到这种只会重启大法。直到有一天，重启也不行了。记得当时比较忙，就先搁置在那，几个月后再上去一看，数据突然就都不见了。而且最无语的是，当时的我居然一点备份意识都没有，数据就放心地存在服务器上。具体是服务器过期了还是啥原因我也记不清了，总之所有之前写过的文字都已然灰飞烟灭，随之而去的是我整个大学前两年的回忆……

打那之后，我所有的文章都发布在了博客园和 CSDN 上，虽然很省心，但心中有一个念头始终没有消失过：我还是一定要有一个只属于自己的个人博客。至于为什么，我也说不太上来。我承认我有装逼的成分，但更重要的是，博客似乎总能给我一种安全感、归属感。虽然域名变了，服务器变了，但对我来说，这里还是当时的那个地方。在纷繁嘈杂的生活中，仍有这么一小块地方，能够容纳自己内心深处最柔软的东西，记录一些所学所读所思所想，确乎是一件十分幸福的事了。有朋友和我说，现在人们都不怎么玩博客了。确实，相比公众号或 Vlog 这类，博客已经不新鲜了，但我就是对这个东西情有独钟啊。也不是为了流量关注度啥的，恰恰相反，我需要的正是这样一个半私密的小小空间。在这个不太容易被搜索引擎抓到的世界里，我便可以是国王。

正如你所见，在本篇之前的文章，都是笔者过去在其他平台上发表过的，现已全部完成迁移，并设置成了他们原本的发表时间。而本篇文字的尴尬之处就在于，明明是在这个博客上写的第一篇，在归档的时间线上却又不是第一篇。姑且就命作《不是首记的首记》吧，哈哈哈。

P.S. GitHub + Hexo 真的太香了。
