概念

  • 完全二叉树：若设二叉树的深度为 h，除第 h 层外，其它各层（1～h-1）的结点数都达到最大个数，第 h 层所有的结点都连续集中在最左边。
  • 用数组表示（下标从 1 开始），则有：
    • arr[i] 的左孩子是 arr[2*i]，右孩子是 arr[2*i+1]；
    • arr[i] 的父节点是 arr[i/2]。
  • 大顶堆：每个结点的值都大于或等于其左右孩子结点的值。
  • 小顶堆：每个结点的值都小于或等于其左右孩子结点的值。

计算机启动过程

发表于 2020-09-07 分类于 学习笔记
本文字数： 2.3k 阅读时长 ≈ 2 分钟

对于经典的 BIOS 技术，整个启动过程主要分为四个阶段。

第一阶段：BIOS

计算机的主板上有一块 ROM 芯片，开机程序被刷入芯片，计算机通电后，第一件事就是读取它。这块芯片里的程序叫做“基本输入输出系统”（Basic Input/Output System），简称为 BIOS。在 BIOS 内有底层 I/O 软件，包括读键盘、写屏幕、磁盘 I/O 以及其他过程。如今，它被保存在闪存中，是非易失性的，当 BIOS 中发现错误时，可以由操作系统进行更新。

事实上，这本书并不像历史教科书那样，按照时间线叙述一个个重要的历史事件。毕竟作为一本“简史”，也不可能做到那样系统、全面。正如高毅教授在序中所说，这本书已经很难说还是通常意义上的“历史”了，它更多是“哲学”，包含了很多人生的哲理。而各个历史事件则倾向于作为本书的线索和例证，用以论证作者深刻而前卫的历史观与世界观。阅读过程中，我常惊叹于作者知识之渊博，各学科领域的运用信手拈来，巧妙而得当。再加之作者生动的文笔和丰富的想象力，更为本书增添了许多文学性，并引人深思。

北航大学生活动发布与社交平台-UniActive

发表于 2020-07-19 分类于 项目
本文字数： 273 阅读时长 ≈ 1 分钟

目前北航的学术讲座、博雅课堂、社团活动和各个学生会组织的活动并没有一个统一的发布平台和入口，学生、老师及活动组织者需要通过不同的网站进行注册报名，并且这些网站也没有相关的日程提醒功能，也无法让活动参与者对组织者提供反馈。我们旨在用一个带有网页端的平台，来改变这一现状，方便北航师生的课余生活。

本系统为面向北航全校师生的活动发布、管理和社交平台，目的旨在方便全校的活动组织者和参与者，在活动发布、宣传通知、日程提醒和参与层面，给予一个统一的发布平台，并基于推荐算法为广大师生提供当前北航正在进行的、人气高的或用户感兴趣的活动。
